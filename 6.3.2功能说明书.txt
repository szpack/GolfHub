  发布日期：2026-03-05 ｜ 技术栈：Vanilla JS + Canvas，无需构建

  ---
  一、产品概述

  GolfOverlay 是一款运行在浏览器中的高尔夫赛事直播辅助工具，无需安装，打开 index.html 即可使用。核心功能是在 Canvas
  上实时渲染两种透明角标：

  - Shot Overlay（击球信息版） — 展示当前洞号、Par、击球类型、距旗杆距离
  - Scorecard Overlay（计分卡信息版） — 展示 18 洞成绩总览

  渲染结果可导出为透明 PNG，用于 OBS 直播叠加或视频后期。

  三、核心功能模块

  3.1 多球员管理

  ---
  3.2 18 洞成绩录入

  成绩模型

  每洞存储 delta（相对标准杆差值）：

  ┌───────┬───────────────────┬──────────────┐
  │ Delta │       名称        │     颜色     │
  ├───────┼───────────────────┼──────────────┤
  │ ≤ −2  │ Eagle / Albatross │ 紫色 #7c3aed │
  ├───────┼───────────────────┼──────────────┤
  │ −1    │ Birdie            │ 红色 #C0392B │
  ├───────┼───────────────────┼──────────────┤
  │ 0     │ Par               │ 蓝色 #1a5bb5 │
  ├───────┼───────────────────┼──────────────┤
  │ +1    │ Bogey             │ 绿色 #2e7d32 │
  ├───────┼───────────────────┼──────────────┤
  │ +2    │ Double Bogey      │ 灰色 #9e9e9e │
  ├───────┼───────────────────┼──────────────┤
  │ +3    │ Triple Bogey      │ 深灰 #555555 │
  ├───────┼───────────────────┼──────────────┤
  │ ≥ +4  │ 超出              │ 黑色 #111111 │
  └───────┴───────────────────┴──────────────┘

  录入方式

  - Delta 大按钮（蓝色圆角卡）：点击弹出选分弹窗，范围 −6 至 +12
  - ± 微调按钮：点击 − / + 对当前 delta 逐格调整
  - 选分弹窗：滚动列表，每行色块对应成绩名称，自动滚动至当前值
  - 清除本洞：弹窗顶部 "Clear" 选项，清空本洞所有数据

  Par 设置

  - 右侧面板 Hole Banner 内联显示 Par 3 / 4 / 5 三个按钮
  - 可在 Settings → Course Par 中一键重置所有洞为 Par 4

  ---
  3.3 击球序列管理

  每洞支持记录多次击球，数量由 Par + delta 自动计算（Gross 杆数）。

  杆进度按钮

  成绩录入后，Shot 区域显示数字色块按钮（1、2、3…），点击直接跳转至该杆：

  ┌──────────┬──────────┐
  │   状态   │   样式   │
  ├──────────┼──────────┤
  │ 当前杆   │ 白底绿字 │
  ├──────────┼──────────┤
  │ 已完成杆 │ 半透明白 │
  ├──────────┼──────────┤
  │ 未来杆   │ 微亮灰   │
  └──────────┴──────────┘

  杆导航

  - ◀ PREV / NEXT ▶ 按钮：循环切换（首杆←→末杆）
  - 键盘 ← / ,：上一杆；→ / .：下一杆
  - 键盘 H：进入下一洞

  击球类型（两排按钮）

  第一排 — 动作类型：

  ┌───────┬──────────────────────────────┐
  │ 类型  │             说明             │
  ├───────┼──────────────────────────────┤
  │ TEE   │ 开球（Tee Off）              │
  ├───────┼──────────────────────────────┤
  │ APPR  │ 上果岭（Approach）           │
  ├───────┼──────────────────────────────┤
  │ LAYUP │ 送上（Layup，Par 5 第 2 杆） │
  ├───────┼──────────────────────────────┤
  │ CHIP  │ 短切（Chip）                 │
  ├───────┼──────────────────────────────┤
  │ PUTT  │ 推杆（Putt）                 │
  ├───────┼──────────────────────────────┤
  │ PROV  │ 暂定球（Provisional）        │
  └───────┴──────────────────────────────┘

  第二排 — 结果类型：

  ┌────────────┬────────────┐
  │    类型    │    说明    │
  ├────────────┼────────────┤
  │ FOR BIRDIE │ 小鸟推杆   │
  ├────────────┼────────────┤
  │ FOR PAR    │ 标准杆推杆 │
  ├────────────┼────────────┤
  │ FOR BOGEY  │ 柏忌推杆   │
  └────────────┴────────────┘

  - AUTO / MANUAL 徽章：自动判断类型时显示绿色 AUTO，手动选择后显示橙色 MANUAL
  - 自动逻辑：按 Par 值和杆序自动推断类型，支持 Par 3/4/5

  距旗杆距离（To Pin）

  - 每杆独立存储距离（单位：yds）
  - Shot 0（开球）存储全球员共享的洞长度

  3.4 总杆统计

  ┌────────┬─────────────────────────────┐
  │  模式  │            说明             │
  ├────────┼─────────────────────────────┤
  │ To Par │ 显示相对标准杆（如 −3、+1） │
  ├────────┼─────────────────────────────┤
  │ Gross  │ 显示实际总杆数              │
  └────────┴─────────────────────────────┘

  切换按钮位于左侧 nav 区下方，同时影响：
  - 18 洞导航卡内显示的成绩数字
  - Scorecard Overlay 中各洞分数格的显示格式

  特别说明：TOT 列（总杆）在计分卡上始终显示 Gross，不受 To Par 模式影响。

  Total Display（总成绩角标）

  - 右侧面板 ptool 区的 Total 开关控制
  - 开启后，Shot Overlay 右上角额外显示全场累计总成绩色块

  ---
  3.5 洞导航

  左侧面板底部的 18 洞导航区，每洞显示洞号、Par 值、当前成绩（色块卡片）。

  导航卡片

  - 点击任意洞号卡：切换至该洞，同时清除汇总状态
  - 当前洞绿色边框高亮 + 底部箭头指示

  F / B / T 统计卡

  ┌────────┬────────────────────────────┐
  │ 统计卡 │            含义            │
  ├────────┼────────────────────────────┤
  │ F      │ 前 9 洞（Holes 1–9），Out  │
  ├────────┼────────────────────────────┤
  │ B      │ 后 9 洞（Holes 10–18），In │
  ├────────┼────────────────────────────┤
  │ T      │ 全 18 洞，Total            │
  └────────┴────────────────────────────┘

  点击统计卡：
  - 对应统计卡高亮（绿色边框）
  - Scorecard Overlay 切换至对应汇总视图（F9 / B9 / 全场）
  - 再次点击洞号卡或 NEXT 进入下一洞，自动退出汇总视图

  ---
  3.6 Scorecard Overlay（计分卡信息版）

  由左侧 ptool 区的 SC 开关控制显示。

  视图范围

  ┌──────┬──────────────────────────────────────┐
  │ 选项 │                 说明                 │
  ├──────┼──────────────────────────────────────┤
  │ F9   │ 仅显示 Holes 1–9（含 OUT 汇总）      │
  ├──────┼──────────────────────────────────────┤
  │ B9   │ 仅显示 Holes 10–18（含 IN 汇总）     │
  ├──────┼──────────────────────────────────────┤
  │ 18H  │ 显示全 18 洞（含 OUT、IN、TOT 三列） │
  └──────┴──────────────────────────────────────┘

  范围选项 F9 / B9 / 18H 作为 radio 按钮，与 SC 开关同行内联显示，勾选 SC 后可见。

  v6.3.2 修复：切换 F9/B9/18H 时立即清除统计卡汇总状态（scorecardSummary=null），确保 Overlay 立即重新渲染到正确范围。

  Canvas 渲染结构

  ┌─────────────────────────────────────────────────────────────┐
  │  绿色标题行：HOLE / 球员名（可选） │ 1 2 3 … │ OUT │ … │TOT │
  ├───────────────────────────────────────────────────────────  │
  │  淡绿 PAR 行：PAR │ 3 4 4 … │ 36 │ 4 5 … │ 36 │ 72         │
  ├─────────────────────────────────────────────────────────────┤
  │  白色 SCORE 行：SCORE │ 🟦 🟥 … │ OUT │ … │ IN │ TOT 总杆  │
  └─────────────────────────────────────────────────────────────┘

  - 成绩格：圆角色块 + 白色字体
  - 当前洞及未来洞：留空（—），不填写成绩
  - 洞 1 时（无任何完成洞）：计分卡不显示
  - OUT / IN：白底黑字汇总框
  - TOT：始终显示 Gross 总杆

  可选配置（Settings）

  - Show Player Name：将计分卡标题行 "HOLE" 替换为球员全名
  - Reset Scorecard Position：重置覆盖层至屏幕中央

  ---
  3.7 Shot Overlay（击球信息版）

  由 ptool 区的 Shot 开关控制显示。尺寸：490 × 132（基准 1920px）。

  布局

  ┌──────────────┬──────────────────────────────────────┐
  │ 左栏         │ 右栏                                  │
  │ HOLE 7       │  球员姓名（34px，粗体）               │
  │ PAR 4        │  击球类型（如 APPROACH）              │
  │ Par 3/4/5    │  距旗杆：156 yds                      │
  │              │  [总成绩色块（可选）]                  │
  └──────────────┴──────────────────────────────────────┘

  总成绩显示逻辑

  - 过程态（TEE / APPR / LAYUP 等）：计前 N−1 洞（不含当前洞）
  - 结果态（PAR / BIRDIE / BOGEY 等）：计入当前洞成绩

  3.11 导出系统

  分辨率选择

  ┌───────┬─────────────┐
  │ 选项  │  实际尺寸   │
  ├───────┼─────────────┤
  │ 4K    │ 3840 × 2160 │
  ├───────┼─────────────┤

  │ 1080P │ 1920 × 1080 │
  └───────┴─────────────┘

  覆盖层不透明度

  导出区 Opacity 滑块（0–100%），影响最终 PNG 中覆盖层的整体透明度。

  四种导出按钮

  ┌───────────────────┬──────────────────────────────────────────────┐
  │       按钮        │                     输出                     │
  ├───────────────────┼──────────────────────────────────────────────┤
  │ 🖼 Shot PNG        │ 当前帧 Shot Overlay 单张透明 PNG             │
  ├───────────────────┼──────────────────────────────────────────────┤
  │ 📊 Scorecard PNG  │ 当前帧 Scorecard Overlay 单张透明 PNG        │
  ├───────────────────┼──────────────────────────────────────────────┤
  │ 📦 Hole Shots ZIP │ 当前洞所有杆位的 Shot PNG 打包为 ZIP         │
  ├───────────────────┼──────────────────────────────────────────────┤
  │ 📦 18 SC ZIP      │ 18 洞每洞完成状态的 Scorecard PNG 打包为 ZIP │
  └───────────────────┴──────────────────────────────────────────────┘

  批量导出显示进度条（底部），完成后自动下载 ZIP。

  文件命名规则

  {Course}_{Player}_H{洞号}_S{杆号}_{类型}_{模式}_{分辨率}.png
  示例：Masters_Pack_H07_S02_APPR_topar_2160p.png

  五、New Round（新一轮）

  点击右上角 New… 按钮，弹窗提供两个选项：

  - Clear this round's scores（默认勾选）：清空所有球员所有洞的成绩、击球记录
  - Reset course Par to 4：将全部 18 洞 Par 重置为 4

  确认后同时清空所有球员及 byPlayer 数据，界面回到初始状态。
